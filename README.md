## 2、技术选型

（1）前台数据处理/交互/组件化：vue2、vue-router、vuex、vue-lazyload、element-ui、swiper

（2）前后台交互：Ajax请求（axios、async/await）、mock数据

（3）模块化开发：ES6、Babel

（4）项目构建：webpack、vue-cli3、eslint

（5）css与编译器：scss
